Purpose: Deep learning-based applications are commonly used in the medical sphere, especially image classification; thus, deep learning has great potential to impact image-based specialties such as radiology. Deep learning may have the potential to address existing issues in sinus radiography. Because an imbalanced dataset causes problems in radiographic classification performance, Generative Adversarial Network (GAN) usage is proposed to improve deep learning performance by generating part of the insufficient data.

GAN is a model for image generation to reconstruct a similar image to input image. GANs working structure is that two networks compete with each other. The discriminator network competes with the generator and tries to distinguish between the realistic fake images coming from the generator, which samples z vector from a latent space to generate, and the real samples. GANs learn to create realistic images that are similar to the input images but not present in the input dataset. Following the tic-toe game approach, the discriminator and generator are enhancing their abilities simultaneously. This is because generator enforces to learn how to generate realistic images according to discriminator’s level of distinguishing power that is arising empirically during the train.

Approach: Three specialists assessed images of Waters’ view radiographs and categorized them into four classes to prepare input datasets. Subsequently, the diagnosed images were resized to 1024 × 1024 pixels through image processing. Next, GANs were used to provide sufficient images for each class. PGAN was utilized for our experiment. GAN implementations were performed twice sequentially to prepare a large amount of valuable synthetic data. An estimation for the performance of supervised learning models with imbalanced and balanced datasets were demonstrated.

Results: With the fake images, we have done PNS image classification. The performance in which the balanced dataset trained surpasses the performance with imbalanced dataset.
